## ğŸ”§ Key Formulas (v2.1)

### CPR Calculation
```
Pivot Point (PP) = (High + Low + Close) / 3
Bottom CPR (BC) = (High + Low) / 2
Top CPR (TC) = (PP - BC) + PP
[If TC < BC: Swap them]
```

### Gap Percentage
```
Gap % = |Today's Open - Previous Close| / Previous Close Ã— 100

Example: Gap 54, Prev Close 25,819
Gap % = 54 / 25,819 = 0.21%
Bucket: 0-0.5%
```

### Candle Classification
**Thresholds:**
- Strong body: > 70th percentile
- Weak body: < 30th percentile
- Based on rolling 20-day window
